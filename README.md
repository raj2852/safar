# safar
Safar is a travel website which has certain features that make it stand out from the rest of the travel websites in the market. These features are:

1.It has a unique one roofed system, where you can find information about all the 29 states under a single platform.
2.It has detailed information about all the 29 Indian states. We don't want you to miss any bounties available in your destination.
3.To enable a wide range of images for the destination, we have embedded links to the google images gallery.
4.The important spots are mentioned as '#have_to_visit' and the less important ones as 'other attractions'.
5.To provide you with a facility for cross-checking of data and keep you updated about the recent instances we have embedded links to the official govt. tourism pages of each state.
